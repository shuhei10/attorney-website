# 青葉法律事務所 - 和モダン × プレミアム ウェブサイト

`ok-maru.jp` のデザインをベースに、さらに「和モダン」と「プレミアム感」を追及した法律事務所のウェブサイトです。

## 特徴
- **洗練された和モダンデザイン**: Noto Serif JPとNoto Sans JPを使い分け、伝統と現代性が融合したタイポグラフィを実現。
- **視認性とアクセシビリティ**: 料金表、アコーディオン、フォーム等の全要素において、高いコントラストと読みやすさを確保。
- **高品質なAI生成画像**: 各取扱業務の専門性を象徴するカスタム画像を配置。
- **GSAPアニメーション**: 上品かつスムースなスクロール演出によるプレミアムなユーザー体験。

## 開発スタック
- HTML5 / CSS3 (Vanilla CSS)
- JavaScript / GSAP (TweenMax)
- ScrollTrigger

## プロジェクトの背景
既存のサイトを大規模に刷新し、信頼性と専門性を視覚的に訴求するデザインへとアップデートしました。特に「文字の読みやすさ」と「情報の整理」にこだわり、ユーザーが迷わず相談へと進める構成にしています。
